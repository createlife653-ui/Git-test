# 精製の原則

## 核心となる考え
精製とは、種子周辺の果肉層をいかに除去するかだけでなく、その過程で微生物の働きをコントロールし、風味をデザインすることである。

## 客観的な事実

### Inoculation / 接種
- **定義**: 精製プロセスに意図的に特定の微生物（酵母、バクテリア）を追加すること
- **目的**: 発酵の方向性をコントロールし、再現性のある風味を作る
- **使われる微生物**: 
  - 酵母（Saccharomycesなど）→ アルコール発酵、エステル香（花・フルーツ）
  - 乳酸菌 → 乳酸発酵、ヨーグルト状の酸味、クリーミーさ
- **使用情景**: アナエロビック（嫌気性）発酵、炭酸マセレーションなど

### Infused / 浸漬（インフェーズド）
- **定義**: 果実全体（果肉+種子）を発酵槽に漬け込み、果実自体が持つ微生物相を活用する精製方法
- **特徴**: 「接種」ではなく、果実表面や環境に naturally に存在する微生物を利用
- **プロセス**: 水に浸けた状態で発酵を進める → 微生物が果肉の糖分を消費しながら風味化合物を作る
- **Natural / Dry Natural との違い**: Naturalは乾燥過程で発酵させるが、Infusedは水に浸けた状態（水洗式の工程に近い）で発酵させる

## 主観的な感じ方
- **Inoculation（接種）されたコーヒー**: よりクリーンで意図的な発酵香が感じられることが多い。再現性が高く、特定の香り（ワイン、フルーツなど）が明確に出る傾向
- **Infused（浸漬）コーヒー**: 自然な複雑味。果肉由来の甘みと発酵による風味の層が重なる。ややワイルドで、独特の個性が強く出ることがある

## 精製方法のスペクトル

```
Washed ──────── Infused ──────── Natural ──────── Inoculated
クリーン          複雑                フルーティー          再現的
```

## 今日の名言
> 「精製とは、微生物との共作業である。」

## 賛否両論 / Debate

### Inoculation（接種）への賛否

| 肯定的意見 | 批判的意見 |
|-----------|-----------|
| 再現性が高くなる | 「人工的」「不自然」とされる |
| 風味を意図的にデザインできる | テロワール（産地の個性）が失われる |
| 失敗リスクが減る | 特定の風味が均一化する（コーヒーの多様性が減る） |
| 安定した品質が提供できる | 微生物の多様性が損なわれる |

### Infused / Natural 発酵への賛否

| 肯定的意見 | 批判的意見 |
|-----------|-----------|
| 産地の個性（テロワール）が表現できる | 再現性が低い |
| 自然で複雑な風味層が生まれる | 失敗リスクが高い |
| 微生物多様性を尊重できる | 品質管理が難しい |

## ワイン業界での扱い

**結論から言うと、ワインの世界でも「何を足していいか」については非常に厳格なルールがあり、コーヒーのインフューズドに近い手法は、実は「ワイン」とは名乗れないことが多い。**

### ワインのルール：何を足してもいいわけではない

| 禁止されている | 認められている |
|---------------|---------------|
| 発酵中に他の果物（桃・イチゴなど）を入れる | ブドウのみを原料とする |
| 人工的な香料を添加する | 酵母の添加（発酵を確実に進めるため） |
| 外部からの香りを定着させる | オーク樽による熟成（バニラ・スパイス香） |

もし桃やイチゴなどの香りを足した場合、それは「ワイン」ではなく：
- 甘味果実酒
- サングリア
- フレーバードワイン

という**別ジャンル**になります。

### テロワール（土地の個性）が絶対的な価値観

ワインファンが最も大切にするのは**「この土地のブドウだから、この味がする」**という価値観。

- 「他のものを混ぜて味を作った」と分かれば、ワインとしての価値は暴落
- 土地の味を守るために、添加を厳しく制限してきた歴史がある

### コーヒーとの決定的な違い

| ワイン | コーヒー（現状） |
|--------|-----------------|
| 土地の味を守るために、添加を制限 | 「驚き」や「分かりやすい美味しさ」を優先 |
| 数百年の歴史でルールが確立 | 実験的精製が盛り上がる過渡期 |
| 「ブドウの力だけで勝負」が美学 | これまで禁じ手だった「添加」に踏み出す |

### あなたの感じた「違和感」の正体

「砂糖を勝手に加えられているような気分」や「コーヒー離れしている」と感じるのは、**ワインでいうところの「ブドウの力だけで勝負していないことへの違和感」**と同じ、非常に真っ当な感覚。

これは農作物の限界を超えて、人間がデザインしすぎた味に対する自然な反応です。

### 楽しみ方の提示

こうした背景を知った上で：
- **伝統派の楽しみ方**: 産地の個性を味わう
- **アトラクションとしての楽しみ方**: 「あえてこの不自然なほどの桃感を楽しんでみよう」と割り切って味わう

どちらも正解。自分が何を飲んでいるかを理解していれば、両方の楽しみ方ができます。
