# コロンビアコーヒー Q&A

## 産地・地理

### Q: 主な産地の読み方は？
**A:**
- Huila（ウィラ）
- Tolima（トリマ）
- Nariño（ナリーニョ）
- Cauca（カウカ）
- Antioquia（アンティオキア）
- Caldas（カルダス）
- Risaralda（リサラルダ）
- Quindío（キンディオ）

### Q: 熱帯山岳気候と熱帯気候の違いは？
**A:**
- **熱帯気候**（平地）：年間を通じて高温（25-30℃以上）、昼夜の温度差が小さい
- **熱帯山岳気候**（高地）：標高が高いため気温が低い（16-22℃）、昼夜の温度差が大きい、年間を通じて安定

コロンビアは「赤道直下だけど標高が高いから、高温すぎずコーヒーにちょうど良い気候」

### Q: どこの産地が標高が高い？
**A: 高い順**
1. **ナリーニョ**：最高、2,000m超えも多い
2. **カウカ**：高い、1,700-2,000m
3. **ウィラ**：高い、1,600-2,000m
4. **トリマ**：やや高い、1,600-1,900m
5. **コーヒートライアングル**（カルダス、リサラルダ、キンディオ）：中くらい、1,200-1,800m
6. **アンティオキア**：低め、1,200-1,800m

高いほど酸味が明るく、風味が複雑になる

### Q: 市場に出回る地域とレアな地域は？
**A:**

**市場に出回る（一般的）**
- コーヒートライアングル（カルダス、リサラルダ、キンディオ）
- アンティオキア
- ウィラ

**レア・高品質（スペシャルティ専門店）**
- **ナリーニョ**★★★：標高最高、高品質、評価高い
- **トリマ**★★：山岳地帯、小規模生産者
- **カウカ**★★：高標高、複雑な風味

**見分け方**：
- 「コロンビア」だけ→ブレンド（トライアングルなど）
- 産地名が明記→スペシャルティ
- 「ナリーニョ」→特に高品質、レア

### Q: 年平均気温は時期で変わる？
**A: ほぼ変わらない**
- コロンビアは赤道直下なので年間を通じて気温は安定
- 「夏も冬もない」
- ただし**標高による差**は大きい（1,000mで22℃、2,000mで15℃）

### Q: 風は強い？
**A: コーヒー栽培地は風が穏やか**
- コーヒーは強風が苦手
- 風が強すぎない谷間や斜面の中腹を選んで栽培
- 防風林（シェードツリー）を植える工夫あり

## 品種・精製

### Q: 品種の相関図は？
**A:**
```
エチオピア原産
    │
    ├─ Typica ─────────────────────┐
    │                              │
    ├─ Bourbon ──┐                 │
    │           │                 │
    │           ├─ Caturra（自然突然変異）──┐
    │           │                          │
    │           └─ Pink Bourbon（変種）     │
    │                                      │
Timor Hybrid ─×─ Caturra → Castillo       │
（自然交雑）  （Cenicafé開発）            │
                                          │
                                  Typica × Bourbon × Timor → Tabi
                                        （Cenicafé開発）

Geisha（エチオピア原産、別系統）
```

### Q: 各品種の特徴は？
**A:**

| 品種 | 特徴 | 味 |
|------|------|------|
| **Caturra** | 古くからある、品質良い | 明るい酸味 |
| **Castillo** | 今の最多、病気に強い | Caturraに近い |
| **Colombia** | Cenicafé開発、安定 | バランス型 |
| **Tabi** | 新しい、耐病性 | 複雑な風味 |
| **Pink Bourbon** | レア、人気 | 甘み、フルーティー |
| **Geisha** | 高級、ジャスミン香 | 花香、トロピカル |

### Q: ナチュラル製法やゲイシャ豆を育てる意識の高い生産地は？
**A:**

1. **ナリーニョ**★★★：標高最高、実験的な加工法、ゲイシャ増加中
2. **ウィラ**★★：「Huila Natural」ブランド、ゲイシャ試験栽培
3. **カウカ**★★：高標高、ナチュラル評価高い

「ナリーニョ」「ウィラ」「カウカ」の高標高地帯が最先端！

### Q: Cenicafé（セニカフェ）って何？
**A: コロンビア国立コーヒー研究所**
- 1938年設立
- 本拠地：Chinchiná（カルダス県、コーヒートライアングル）
- FNCの研究部門
- **Castillo、Colombia、Tabi**などを開発
- 病気（さび病）に強い品種開発のリーダー

## 認証

### Q: 認証の種類と違いは？
**A:**

| 認証 | 内容 | 特徴 |
|------|------|------|
| **Rainforest Alliance** | 環境保護＋労働条件 | カエルの緑ロゴ、大規模農園で多い |
| **UTZ** | 農業管理のトレーサビリティ | 2018年にRainforest Allianceと統合 |
| **Fair Trade** | 公平な取引価格 | 青白ロゴ、小規模生産者の協同組合で多い |
| **Organic** | 化学農薬・肥料不使用 | 3年間の転換期間必要、全体の5%以下 |
| **Bird Friendly** | 鳥類保護 | オーガニック＋日陰樹必須、まだ少ない |

**注意**：認証＝美味しいとは限らない。スペシャルティは認証なしが多い。

## 特殊処理

### Q: 特殊処理の種類は？
**A:**

| 処理 | 方法 | 味 | 人気 |
|------|------|------|------|
| **嫌気発酵** | 密閉容器で発酵 | ワイン風、複雑 | ⭐⭐⭐ |
| **酵母発酵** | 酵母添加 | フルーティー、花香 | ⭐⭐ |
| **ラクティック発酵** | 乳酸菌優位のナチュラル | チーズ風、濃厚 | ⭐⭐ |
| **コールド・ウォッシュ** | 低温で長時間発酵 | クリーン、明るい酸味 | ⭐⭐⭐ |
| **二重発酵** | 発酵を2回に分ける | 非常に複雑 | ⭐ |

**注意**：「特殊処理」＝「美味しい」ではない。失敗すると不味い（発酵臭）、価格も高い。伝統的なWashedも最高品質あり。

## その他

### Q: エメラルドマウンテンはアンデス山脈？
**A: エメラルドマウンテン＝ブランド名**
- 特定の山の名前ではない
- FNCが扱うブランド
- 実際の産地は「アンデス山脈の麓」（特定の産地ではない）
- 「エメラルド」はコロンビアがエメラルド産地として有名なことから

### Q: コロンビア全体の豆の特徴は？
**A:** 明るい酸味、柑橘類、キャラメル、チョコレート、バランスの取れたミディアムボディ、クリーンなフィニッシュ
