# 基本の抽出レシピ（再現性重視）

## 何のためにやるの？
毎回同じ味を安定して出すための「基準レシピ」。ブレを減らして、比較しやすくする。

## 手順

1. **豆とお湯の比率**：1対15（例：豆20gならお湯300ml）
2. **挽き目**：中挽き（やや粗め）
3. **お湯の温度**：88℃
4. **注ぎ方**：1投あたり20ml × 5投で合計100ml（残りは比率で調整）
   - 1投目（蒸らし）：開始〜30秒
   - 2投目：1分45秒
   - 3投目：2分15秒
   - 4投目：2分45秒
   - 5投目：3分30秒

## うまくいったかな？
覚えやすく、無難なコーヒーが安定して出る。迷ったときに戻る「基準点」として使いやすい。
