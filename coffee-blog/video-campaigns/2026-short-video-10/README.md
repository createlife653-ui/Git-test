# coffee-blog 10本ショート動画キャンペーン

既存5記事から10本の縦動画を制作し、Instagram Reels / YouTube Shorts / TikTokへ投稿するための実行管理フォルダです。

## ゴール

- 10本のマスター動画を作る。
- 各動画を3平台へ投稿し、合計30投稿を完了する。
- 全投稿にブログ誘導CTAを入れる。
- 投稿URLと48時間後の初期数値を記録する。

## 制作方針

- 撮影なし、顔出しなし、声なし。
- AIイラストまたはイメージ映像、字幕、BGMで構成する。
- 編集はCanva中心。TapNowは補助素材作成で明確な利点がある場合だけ使う。
- 1本は20-45秒、9:16、1080x1920、MP4で書き出す。
- AI生成ラベルが選べる平台ではオンにし、投稿文にもAIイラスト使用を明記する。

## 使うファイル

- `campaign-board.csv`: 10本全体の進行、投稿URL、CTA先。
- `posting-calendar.csv`: 投稿予定日と実績。
- `metrics.csv`: 48時間後の平台別数値。
- `workflow.md`: 作業手順。
- `templates/`: 各動画ファイルのテンプレート。
- `videos/vXX-*/`: 各動画の台本、画像プロンプト、投稿文、QA、数値。

## 進め方

1. `v01` を試作し、字幕量・絵柄・Canvaテンプレを決める。
2. `v02` 以降は同じ型で量産する。
3. 投稿後に `campaign-board.csv` へURLを入れる。
4. 48時間後に `metrics.csv` と各動画の `metrics.md` を更新する。
5. 10本完了後、このREADMEへ振り返りを追記する。
